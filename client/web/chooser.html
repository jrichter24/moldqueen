About MoldQueen

Control a Mould King 13112 RC excavator from a Raspberry Pi over a reverse-engineered BLE protocol, through a clean WebSocket API.

Disclaimer

Credits

  • J0EK3R/mkconnect-python (MIT) — the MouldKingCrypt cipher (ported byte-for-byte here) and the MK4/MK6 protocol groundwork. Our hubs turned out to be the MK4 12-channel nibble variant (found by capturing the MK+tech app); the codec is verified byte-exact against the app's adverts.
  • BrickController2 — additional protocol reference.

How MoldQueen was built

MoldQueen was built with help from AI coding assistants. Early work used Claude (Fable), later work used Claude Opus 4.8. All design decisions and the code review were done by the human developer.
